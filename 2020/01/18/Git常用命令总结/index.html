Clone

知道一个远程库的git地址，直接通过 git clone 就代码库的地址就将代码下载下来。

注意：这个时候会在本地自动创建一个 origin 的远程连接，这个分支默认的。

fetch：将远程库的某个分支拉到本地就形成新的分支

这样好处：在于每次最新开发都可以从远程库上develop分支的最新的代码开始开发，并在本地建新的分支，到时候MR的时候，将dev1分支提到远程的develop分支。
