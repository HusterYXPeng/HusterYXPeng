不同人修改同一文件的不同区域

1、本地修改代码前使用 git pull

在每次修改代码中，先执行 git pull 命令，将远程分支与本地分支同步，将本地分支做一个更新。

直接 git pull 即可，无需任何其他命令。

2、解决步骤：同上。
